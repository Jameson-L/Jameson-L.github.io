Interestingly, all of the generated outputs look extremely similar. This likely occurs because when the denoiser is trained only to map pure noise directly to clean digits, it learns to always produce the "average" digit. The absence of structured noise in the input means the model doesn't receive any useful signal for generating different digits, leading it to output a blurry shape that vaguely resembles a digit regardless of the input noise.
